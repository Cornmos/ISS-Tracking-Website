ABOUT

For my Universtiy unit Human Computer interactions, I developing a webapp that tracks Satellites in real time by using Maps. This application takes information generated by NASA about the International Space Station (ISS) orbiting us currently.

The location of the satellite is displayed on a map in real time and additional information about the satellite is displayed as well, like Lang, Long Altitude and current speed.

This information is gathered from the NASA ISS web API and using JSON to map the information to this site
